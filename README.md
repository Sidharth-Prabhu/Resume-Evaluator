# AI Resume Evaluator CLI

A robust, Python-based command-line tool to evaluate your resume against specific job descriptions. 

## Features
- **Automatic Resume Detection**: Automatically finds PDF files in the project directory.
- **Job Matching**: Evaluates your resume against a database of job descriptions.
- **TF-IDF & Skill Matching**: Uses advanced text analysis to provide an accurate compatibility score.
- **Actionable Feedback**: Provides specific recommendations and identifies missing skills.

## Setup

1. **Navigate to the directory**:
   ```bash
   cd resume-evaluator-cli
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Place your resume**:
   Make sure your resume (PDF format) is in the `resume-evaluator-cli` folder.

## Usage

Run the program using Python:
```bash
python main.py
```

Follow the interactive prompts to select your resume and the job position you're applying for.
